# Privacy Policy - Spotify Unfollowers Extension

**Effective Date:** January 1, 2026

## Overview

Spotify Unfollowers ("the Extension") is a browser extension that helps users identify who doesn't follow them back on Spotify. This privacy policy explains how the Extension handles user data.

## Data Collection

The Extension **does not collect, store, or transmit any personal data** to external servers. All operations are performed locally within your browser.

### What the Extension Accesses

1. **Spotify Profile Data (Temporary)**
   - Your Spotify user ID
   - Display name and profile avatar URL
   - Following list
   - Followers list

2. **Data Processing**
   - The Extension accesses Spotify web pages only when you actively use it
   - All data is processed locally in your browser's memory
   - No data is sent to third-party servers

### What the Extension Does NOT Collect

- Login credentials
- Personal messages or communications
- Financial or payment information
- Browsing history outside of Spotify
- Analytics or usage data
- Location information

## Data Storage

- All scan results are stored temporarily in Chrome's local storage
- Data is automatically cleared when you uninstall the Extension
- No persistent identifiers are maintained
- No cookies are set or tracked

## Data Sharing

The Extension **does not**:
- Share data with third parties
- Sell or rent user data
- Use data for advertising or marketing purposes
- Transmit data to external servers

## Third-Party Services

- The Extension loads fonts from Google Fonts (fonts.googleapis.com)
- This is for UI styling only and does not involve user data collection
- Font requests are standard browser operations and do not constitute data sharing

## Security

- All operations occur locally within your browser
- The Extension uses Manifest V3 with strict Content Security Policy
- No remote code execution is performed
- No `eval()` or dynamic code execution

## Permissions

The Extension requests minimal permissions:
- **activeTab**: To interact with the currently active Spotify tab
- **storage**: To save scan results temporarily within your browser

## Children's Privacy

The Extension does not knowingly collect data from users under 13 years of age.

## Changes to This Policy

Any changes to this privacy policy will be updated in the Extension's source code and accompanied by an updated effective date.
